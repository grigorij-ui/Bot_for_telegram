# Telegram бот для бара

## Что умеет
- Главное меню: `Меню бара` и `Наш адрес`.
- Раздел `Наш адрес` отправляет ссылку и адрес.
- Раздел `Меню бара` -> выбор: `Коктейль` или `Шоты`.
- После выбора позиции создается уникальный номер заказа от `00001` до `10000`.
- Карты для оплаты выдаются по очереди из списка `CARDS` (циклически).
- Покупатель может `Подтвердить оплату` или `Отменить заказ`.
- Админу приходит заявка на проверку оплаты (`Подтвердить` / `Отклонить`).
- При подтверждении: бармен получает номер и позицию заказа.
- При отклонении: покупатель получает кнопку `Повторить попытку`, которая ведет в главное меню.

## Установка
```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
```

## Настройка
Откройте `bot.py` и заполните:
- `BOT_TOKEN`
- `ADMIN_ID`
- `BARMEN_ID`
- список `CARDS` (ваши 5 карт)

## Запуск
```bash
python bot.py
```

## Важно
- Счетчик заказов и индекс карты сохраняются в `state.json`.
- После номера `10000` новые заказы не создаются (по вашему условию).
